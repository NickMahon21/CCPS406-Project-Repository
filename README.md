# Web App for BizPoints Admin Dashboard

Interactive Admin Dashboard
Live link to dashboard: https://ccps406-project-repository-admxn-dashboard.streamlit.app/


Project Structure:
- `main.py` – Streamlit web application
- `api_mock.py` – Mock API simulating backend services
- `user_management.py` – User role assignment logic
- `support_feedback.py` – Ticket and feedback management
- `rewards_management.py` – Point redemption and adjustment logic
- `kpi.py` – KPI aggregation layer
- `tests/` – Unit tests
- `libraries.txt` – App dependencies
